Website Project 2: Energy Tracking System

This project aims to provide accurate data on using electric vehicles as an alternative to the conventional internal combustion engine. The software produced will be focused on providing this data to businesses in South Africa. The team plans on launching an online site that allows businesses to track their current energy usage and receive comparison usage data if electric vehicles are used as an alternative. The current version is just a working prototype. My task was to design the website structure, website styling, energy calculations, unit testing and functionality testing.
