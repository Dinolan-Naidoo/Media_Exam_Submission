Unity Project 1: Perfect Timing

This prototype is an investigation of various path-finding algorithms that can be used to create artificially intelligent NPCs (non-player characters) within games. The aim of the investigation is to have a comparison between different algorithms in order to find the most efficient path-finding algorithm. The criteria for efficiency includes the time taken to find a path, as well as the node exploration efficiency. The investigation is conducted on five different algorithms. These include Breath First Search , Depth First Search , Dijkstraâ€™s algorithm , A* Manhattan and A* Euclidean. The expected final result of this investigation should yield an efficient path-finding NPC that can navigate around various obstacles to reach a target node. Furthermore, the user will be given a clear comparison of the different path-finding algorithms being investigated.
